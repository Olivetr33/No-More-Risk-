RISKER – Local Risk Table Web App
A privacy-first, individually tailored web application for managing, filtering, and archiving customer risk data – all locally in your browser.
No server. No tracking. 100% local.
ABOUT
Local
RISKER is a local, client-side web app for risk management and customer data analysis.
Custom
This project is individually tailored for scenarios where sensitive data must never leave the user's device.
No Cloud
All logic runs in the browser. No server, no external APIs, no tracking.
FEATURES
Local Excel Upload
Import .xlsx or .xls files directly in the browser.
Modern UI
Responsive sidebar menu, sticky footer, dark mode design.
Filtering & Sorting
Filter by LCSM, sort by any column.
Risk Aggregation
Automatically aggregates multiple rows per customer and sums all risk values.
Archiving
Mark customers as "done" and move them to the archive; archived customers are excluded from new uploads.
Clear Data
Remove all table data and optionally the archive, with confirmation dialog.
Privacy-First
No data leaves your device. No tracking, no cookies, no server.
Accessibility
Keyboard navigation, tooltips, and clear button labels.
Customizable Background
Animated, blurred video background possible.
GLOSSARY
Risk Table
Tabular overview of all customers and their risk scores.
Legal Customer Success Manager
Responsible for the legal and operational management of a customer.
Used for filtering and assignment.
Archive
Local list of all customers marked as "done". Archived customers are hidden in the main table and excluded from new uploads.
Aggregation
Multiple rows per customer are combined into one, all risk values are summed.
Completed
A customer entry that has been marked as completed and moved to the archive.
Clear Data
Function to delete all currently loaded table data and optionally the archive.
Local
All data and processing stays in your browser; nothing is sent to a server.
GETTING STARTED
Browser
A modern web browser (Chrome, Firefox, Edge, Safari) is required. No server or backend needed.
Install
Clone or download this repository:
git clone https://github.com/Olivetr33/risker_read-it-.git
BG Video
(Optional) Place your own background.mp4 in the project directory for a custom video background.
Start
Open index.html in your browser. Click "NO MORE RISK!" to launch the app.
Upload
Use the Upload Data button in the sidebar to import your Excel file.
Manage
Filter, sort, and manage your risk data. Use Clear Data to remove all data and/or the archive.
Info
Privacy information and GitHub link are always available in the footer.
EXCEL DATA FORMAT
Columns
Column Name Description Example Value
Customer Number Unique customer number (string or number) 1001
Name Customer name ACME Corp
LCSM Legal Customer Success Manager John Doe
ARR Annual Recurring Revenue (optional) 12000
Total Risk Total risk score (summed on import) 15
Contact Risk Contact risk score (summed on import) 2
Update Risk Update risk score (summed on import) 3
Value Risk Value risk score (summed on import) 4
Contract Risk Contract risk score (summed on import) 1
Objective Risk Objective risk score (summed on import) 5
Important:
You can have multiple rows per customer (e.g. for different risk events).
The app will automatically aggregate (sum) all risk values for each customer.
All columns must be present and named exactly as shown above.
Additional columns are ignored.
